Represents an RGB color value.

Defines an RGB color value with components between 0 and 1.

- 'r': The red component.
- 'g': The green component.
- 'b': The blue component.

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
|  `r` | `number` | Red |
|  `g` | `number` | Green |
|  `b` | `number` | Blue |
