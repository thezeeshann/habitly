```ts
type BlockState = 
  | BlockStateError
  | BlockStatePending
  | BlockStateReady;
```

Represents the state of a design block.

The `BlockState` type is a union of the `BlockStateError`, `BlockStatePending`, and
`BlockStateReady` interfaces, which describe the possible states of a design block.
