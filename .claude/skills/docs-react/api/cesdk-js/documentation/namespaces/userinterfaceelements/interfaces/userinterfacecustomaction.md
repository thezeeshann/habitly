Interface representing a custom action in the user interface.

- `callback`: Function to be called when the action is triggered.
- `label`: Label for the action.
- `iconName`: Name of the icon for the action.

## Properties

| Property | Type |
| ------ | ------ |
|  `callback` | () => `void` | `Promise`\<`void`> |
|  `label` | `string` |
|  `iconName` | [`UserInterfaceCustomActionIconName`](./api/cesdk-js/documentation/namespaces/userinterfaceelements/type-aliases/userinterfacecustomactioniconname.md) |
