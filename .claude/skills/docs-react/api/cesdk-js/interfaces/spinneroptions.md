Represents options for a loading spinner.

The `SpinnerOptions` interface controls the appearance of an indeterminate
loading spinner. The optional `label` is rendered beneath the spinner.

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
|  `label?` | `string` | `string`\[] | An optional caption rendered beneath the spinner (e.g. `'Loading...'`). Strings are passed through i18n and resolve to themselves when no translation is available. |
