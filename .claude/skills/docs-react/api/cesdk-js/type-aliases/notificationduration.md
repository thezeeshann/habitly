```ts
type NotificationDuration = number | "infinite" | "short" | "medium" | "long";
```

Represents the duration of the notification.

The NotificationDuration type defines the possible durations for notifications within the Creative Editor SDK.
Each duration corresponds to a different time span, allowing for flexibility in how long notifications are
displayed to users.
