type AnimationEntry = object;

Configuration options for animations.

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
|  `type` | [`AnimationType`](./api/engine/type-aliases/animationtype.md) | The type of animation to apply |
|  `duration` | `number` | Duration of the animation in seconds |
|  `easing?` | [`AnimationEasing`](./api/engine/type-aliases/animationeasing.md) | Easing function for the animation |
