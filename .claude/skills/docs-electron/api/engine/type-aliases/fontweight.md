```ts
type FontWeight = 
  | "thin"
  | "extraLight"
  | "light"
  | "normal"
  | "medium"
  | "semiBold"
  | "bold"
  | "extraBold"
  | "heavy";
```

Allowed font weights. Mirrors the `@cesdk/engine` (WASM) `FontWeight`
union so a single `Font` is interchangeable across bindings.
