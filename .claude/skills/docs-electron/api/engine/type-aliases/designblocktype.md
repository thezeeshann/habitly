```ts
type DesignBlockType = 
  | DesignBlockTypeShorthand
  | DesignBlockTypeLonghand;
```

The block type IDs for the top-level design blocks. These are the IDs used to create new blocks
using `cesdk.engine.block.create(id)`.
Refer to [DesignBlockTypeShorthand](./api/engine/type-aliases/designblocktypeshorthand.md) and [DesignBlockTypeLonghand](./api/engine/type-aliases/designblocktypelonghand.md) for more details.
