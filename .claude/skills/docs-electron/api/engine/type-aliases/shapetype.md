```ts
type ShapeType = 
  | ShapeTypeShorthand
  | ShapeTypeLonghand;
```

The block type IDs for the shape blocks. These are the IDs used to create new shapes
using `cesdk.engine.block.createShape(id)`.
Refer to [ShapeTypeShorthand](./api/engine/type-aliases/shapetypeshorthand.md) and [ShapeTypeLonghand](./api/engine/type-aliases/shapetypelonghand.md) for more details.
