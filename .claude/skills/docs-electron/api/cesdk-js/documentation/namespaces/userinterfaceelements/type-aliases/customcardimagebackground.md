```ts
type CustomCardImageBackground = object;
```

Type representing an image background for a custom card.

- `url`: The URL of the image.
- `type`: The type of the background, which is 'image'.

## Properties

| Property | Type |
| ------ | ------ |
|  `url` | `string` |
|  `type` | `"image"` |
