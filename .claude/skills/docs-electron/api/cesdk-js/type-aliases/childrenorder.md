```ts
type ChildrenOrder = (
  | ComponentId
  | OrderComponent<ComponentId>)[];
```

Represents the order of children components in a dropdown.

The `ChildrenOrder` type provides a set of values that control the order of children
components in a dropdown. These options include settings for the component ID or
order component.
