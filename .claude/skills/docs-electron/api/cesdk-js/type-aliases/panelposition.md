```ts
type PanelPosition = "left" | "right" | "bottom";
```

This type is used to specify the position of various panels within the user interface,
such as the inspector, settings, and asset library panels.

By setting the position to
`'left'` or `'right'`, you can control the layout and placement of these panels to better
suit the user's workflow and preferences.
