```ts
type VideoSupportDialogOptions = object;
```

Dialog display options for video support check actions.
Allows configuring whether and how the dialog is displayed.

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
|  `show` | `boolean` | Whether to show the dialog |
|  `backdrop?` | `"transparent"` | `"opaque"` | Backdrop style for the dialog - 'transparent' allows interaction with the page, 'opaque' blocks it |
